tema 1- texto
.
Propiedad Valor Inicial Descripción
color black Esta propiedad define el color del texto dentro de un elemento. Puede especificarse usando valores como nombres de colores (red), códigos hexadecimales (#ff0000), valores RGB (rgb(255, 0, 0)), RGBA, HSL, entre otros.
font-family serif (dependiendo del agente de usuario) Define el tipo de fuente que se utilizará para el texto del elemento. Puede especificarse una lista de fuentes, en la que el navegador buscará la primera disponible.
font-size medium Establece el tamaño de la fuente del texto. Puede ser especificado en varias unidades, como píxeles (px), em, rem, porcentaje, entre otros.
font-weight normal Controla el grosor de las letras en el texto. Los valores comunes incluyen normal, bold, lighter, bolder y valores numéricos entre 100 y 900 (por ejemplo, 400 es normal, 700 es negrita).
font-style normal Define el estilo de la fuente, permitiendo usar valores como normal (por defecto), italic (cursiva) o oblique (una inclinación ligera).
font-variant normal Controla la variación del estilo tipográfico, como la utilización de small-caps (letras en mayúsculas pequeñas) o variantes de tipo OpenType. Afecta principalmente a fuentes que admiten variantes tipográficas.
font Ninguno (es una propiedad abreviada) Esta propiedad es una abreviación que agrupa varias propiedades relacionadas con la fuente, como font-style, font-variant, font-weight, font-size, line-height y font-family. Un ejemplo de su uso sería: font: italic bold 16px/1.5 Arial, sans-serif.
text-align left (dependiendo del idioma) Esta propiedad especifica la alineación horizontal del texto dentro de un contenedor. Los valores comunes son left, right, center, justify.
line-height normal Define la altura de la línea de texto, lo que influye en el espacio entre líneas de un párrafo o en elementos con texto. Este valor puede ser una unidad (como px, em) o un número (multiplicando el tamaño de la fuente).
text-decoration none Controla el estilo de decoración del texto, como subrayado (underline), tachado (line-through), sobrelínea (overline), o ninguna decoración (none).
text-transform none Especifica cómo se debe transformar el texto en cuanto a mayúsculas o minúsculas. Los valores comunes son none, uppercase (convertir a mayúsculas), lowercase (convertir a minúsculas), y capitalize (convertir la primera letra de cada palabra a mayúscula).
vertical-align baseline Determina la alineación vertical del contenido en relación con otros elementos o con la línea base del texto. Algunos valores incluyen top, middle, bottom, o valores específicos como sub o super para texto en subíndice o superíndice.
text-indent 0 Establece la cantidad de espacio en blanco que se debe dejar al principio de una línea de texto (por lo general, usado para crear sangrías en párrafos).
letter-spacing normal Controla el espacio entre caracteres de un texto. Se especifica con unidades como píxeles (px) o em.
word-spacing normal Define el espacio entre las palabras de un texto. Similar a letter-spacing, se especifica en unidades como píxeles o em.
white-space normal Controla cómo se gestionan los espacios en blanco dentro del texto, tales como espacios consecutivos y saltos de línea.

tema 2 -Enlaces

caracteristica propiedades Descripción
Tamaño, Color y Decoración Las propiedades de tamaño, color y decoración en CSS permiten controlar la apariencia y disposición de los elementos en una página web. Las propiedades de tamaño incluyen width, height, font-size, padding, margin y border-radius, que definen las dimensiones, el espaciado y el redondeo de los bordes. Las propiedades de color como color, background-color, border-color, box-shadow y text-shadow controlan los colores de los textos, fondos, bordes y sombras. Las propiedades de tamaño, color y decoración en CSS son fundamentales para diseñar y personalizar la apariencia de los elementos en una página web. El tamaño se refiere a las dimensiones y espaciado de los elementos, controlado mediante propiedades como width, height, font-size, padding y margin. El color permite definir los colores de texto, fondo, bordes y sombras utilizando propiedades como color, background-color, border-color y box-shadow. La decoración añade detalles visuales como bordes, sombras, transformaciones y efectos interactivos, utilizando propiedades como text-decoration, border, transform y text-shadow. Juntas, estas propiedades permiten crear un diseño atractivo, coherente y funcional en la interfaz web.
Decoración Personalizada text-decoration: Se utiliza para aplicar decoraciones al texto, como subrayado, tachado o sobrelineado. border: Define el estilo, grosor y color del borde de un elemento. box-shadow: Crea sombras alrededor de un elemento, lo que añade un efecto de profundidad. text-shadow: Aplica sombras al texto. transform: Permite modificar la posición, escala, rotación y otras transformaciones de un elemento. transition: Define las animaciones suaves cuando cambian las propiedades CSS. animation: Permite crear animaciones complejas y dinámicas para los elementos. La decoración en CSS agrega un toque visual a los elementos, permitiendo que la página tenga más estilo y profundidad. Con text-decoration, puedes agregar efectos como subrayados o tachados en texto. Las propiedades de border y box-shadow son comunes para estilizar los elementos y crear una jerarquía visual. Las transformaciones y animaciones permiten modificar la apariencia de los elementos de manera dinámica.

tema 3 - imagenes

caracteristica propiedades Descripción
Anchura y Altura width: Define el ancho de un elemento. height: Define la altura de un elemento. Las propiedades de anchura (width) y altura (height) permiten controlar las dimensiones de un elemento en CSS. El valor de width establece el ancho del elemento, mientras que height define su altura. Estas propiedades pueden tomar valores absolutos (en píxeles, por ejemplo, 200px) o relativos (como un porcentaje 50% del contenedor padre), y son esenciales para organizar el espacio de un diseño. Al usar estas propiedades correctamente, se puede ajustar el tamaño de los elementos según las necesidades del diseño y asegurar que se alineen adecuadamente en la página.
Eliminar el Borde border: Elimina el borde de un elemento configurando su valor a none. Para eliminar el borde de un elemento en CSS, se utiliza la propiedad border, configurando su valor como none. Esta propiedad es útil cuando deseas que un elemento no tenga bordes visibles, incluso si anteriormente se les había aplicado un estilo de borde. Por ejemplo, si tienes un botón con un borde y deseas quitarlo, puedes usar border: none;. Esto asegura que el elemento se muestre sin ningún contorno o línea alrededor de él.

tema 4 - Listas

Propiedad Valor Inicial Descripción
list-style-type disc (círculo sólido) La propiedad list-style-type se utiliza para definir el tipo de marcador (símbolo) que se mostrará antes de cada elemento de una lista (
    ,
      ). Los valores más comunes incluyen disc (un círculo sólido), circle (un círculo vacío), square (un cuadrado), y none (sin marcador). También puede aceptar valores para listas ordenadas, como decimal (números) o lower-alpha (letras en minúsculas). Esta propiedad permite personalizar la apariencia de las listas de manera sencilla, controlando el estilo de los puntos o números que las preceden.
list-style-position outside La propiedad list-style-position controla la ubicación del marcador (punto, número o imagen) en una lista.
list-style-image none La propiedad list-style-image permite reemplazar el marcador predeterminado (generalmente un círculo o número) por una imagen personalizada. Si se establece una imagen de fondo para los elementos de la lista, el valor url("ruta-de-la-imagen") define la imagen que se usará como marcador. Si no se quiere ningún marcador, se puede asignar el valor none. Esto es útil cuando se quiere un estilo de lista más personalizado con imágenes o íconos como puntos de la lista.
list-style none (sin marcador, aunque es una combinación de las tres propiedades anteriores) La propiedad list-style es una forma abreviada que combina las tres propiedades anteriores (list-style-type, list-style-position y list-style-image). Permite definir de manera compacta el estilo de los marcadores de una lista en un solo bloque de código. Por ejemplo, list-style: square inside url('imagen.png'); aplicaría un marcador cuadrado, lo posicionaría dentro del contenido de la lista y usaría una imagen personalizada como marcador. Usar list-style: none; eliminaría completamente los marcadores.

tema 5- Tablas


Propiedad Se Aplica a Valor Inicial Descripción
border-collapse Si deseas que los bordes de las celdas de una tabla se fusionen en un solo borde, puedes utilizar border-collapse: collapse;. Esto elimina los bordes dobles entre celdas, creando una apariencia más limpia. Si prefieres que las celdas tengan bordes separados, puedes usar border-collapse: separate; y controlar el espacio entre las celdas con border-spacing. La propiedad border-collapse se utiliza para controlar cómo se manejan los bordes de las celdas en una tabla. Si se establece en collapse, los bordes de las celdas adyacentes se fusionan en un solo borde, eliminando los bordes dobles entre celdas. Esta opción es útil cuando se desea una apariencia de tabla más compacta, sin líneas duplicadas entre las celdas. Si se establece en separate, cada celda tiene su propio borde, y el espacio entre ellos se puede controlar con la propiedad border-spacing. La propiedad border-collapse controla el comportamiento de los bordes en las celdas de una tabla. Si se establece en collapse, los bordes adyacentes de las celdas se fusionan en un solo borde, lo que da como resultado una tabla con un diseño más compacto, sin bordes dobles entre celdas. Si se establece en separate, los bordes de cada celda permanecen separados, y se puede controlar el espacio entre ellos mediante la propiedad border-spacing.
border-spacing Si has configurado border-collapse como separate y deseas dar más espacio entre las celdas de la tabla, puedes usar border-spacing para definir la distancia entre ellas. Por ejemplo, si configuras border-spacing: 10px;, creas un espacio de 10 píxeles entre los bordes de las celdas. La propiedad border-spacing solo tiene efecto cuando la propiedad border-collapse está configurada en separate. Se utiliza para definir el espacio entre los bordes de las celdas de una tabla. Si deseas agregar un espacio entre las celdas de la tabla, puedes especificar un valor en píxeles o en otras unidades, como em o rem. Establece el espacio entre las celdas de una tabla cuando border-collapse está configurado como separate.
empty-cells Si quieres ocultar las celdas vacías en una tabla para que no se vean bordes en aquellas celdas que no contienen información, puedes usar empty-cells: hide;. Esto es útil cuando tienes tablas con datos opcionales y prefieres que las celdas vacías no ocupen espacio visualmente. La propiedad empty-cells se utiliza para controlar si las celdas vacías en una tabla deben ser visibles o no. Si se establece en show (valor predeterminado), las celdas vacías se muestran con bordes, aunque no contengan contenido. Si se establece en hide, las celdas vacías se ocultan completamente, lo que puede hacer que la tabla se vea más limpia, especialmente si algunas celdas no tienen datos. Controla si se muestran o no los bordes y el fondo en las celdas vacías de una tabla.
caption-side Si deseas que la leyenda de la tabla se muestre debajo de la tabla en lugar de arriba, puedes utilizar caption-side: bottom;. Esto coloca la leyenda en la parte inferior de la tabla, cambiando su orden predeterminado que normalmente está en la parte superior. La propiedad caption-side controla la ubicación de la leyenda o caption de una tabla. Por defecto, la leyenda aparece en la parte superior de la tabla, pero si se desea, se puede mover a la parte inferior de la tabla utilizando este valor. Esto es útil cuando el título de la tabla debe aparecer al final en lugar de al principio, por ejemplo, en tablas con muchos datos o cuando se desea un diseño específico. Especifica la posición del título (caption) de la tabla, ya sea en la parte superior o en la parte inferior de la tabla.
